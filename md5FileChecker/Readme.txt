The purpose of this program is verify that the downloaded ZIP file accompanying md5 text file matches. 

Ex:
apache-cxf-2.7.0 			 -> d957c115b8fbff0e01760660b842fd7e
apache-cxf-2.7.0.zip.md5     -> d957c115b8fbff0e01760660b842fd7e

Steps:

1) Just place the pair of files in a common directory
2) In RunVerify.java specify this directory
3) console will output results
